# 🕒 Digital Clock

A simple and stylish digital clock built using HTML, CSS, and JavaScript.

## 📌 Features

- Displays current hours, minutes, and seconds
- Updates in real time
- Fully responsive and works across devices
- Clean and minimal design

## 🔧 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
